# Crane Automation HMI

## Overview

This project demonstrates a Python-based crane automation system using Modbus TCP communication and a Tkinter Human Machine Interface (HMI).

## Features

- Manual crane movement (X/Y)
- JSON-based motion sequences
- Automatic process execution
- Vacuum control
- Sensor monitoring
- CSV logging
- Position editing through HMI

## Technologies

- Python
- OpenCV
- Tkinter
- Modbus TCP
- Pandas
- JSON

## Note

This repository contains only the Python source code and JSON configuration.

The crane simulation executable is not included. To run the project, start a compatible Modbus TCP simulator or PLC server before launching the application.
